# week01

#### 安装 brew install uv，为什么使用uv

- 管理Python的解释器，替代pyenv
- 管理虚拟环境，替代venv virtualenv
- 管理包依赖，替代pip pip-tools
- 项目全生命周期管理 (替代 `Poetry` / `pipenv`)

#### 使用docker安装并且启动mysql,redis

- 安装docker 和 可视化数据库工具tablePlus
- 在项目中创建`docker-compose.yml`文件内容
- 启动命令：docker-compose up -d，查看启动的容器：docker ps，停止所有服务：docker-compose stop，启动全部服务：docker-compose start

#### 使用uv 初始化，启动fast_web

- `cd fast_web`,`uv init`
- `uv pip list`
- `un add fastapi uvicorn`
- `uv run uvicorn main:app --reload`
